Roles

How It Works?

Cliniva is a template that comes with multiple role management. You can use single template for multiple roles, With template you will get built-in three roles Admin, Doctor & Patient. You can create more as per your requirements. Here are some key features:

  • keyboard_arrow_rightOne Tempalate For All Modules Like Admin, Doctor & Patients.
  • keyboard_arrow_rightAdd Multiple Roles As Per Your Requirements.
  • keyboard_arrow_rightRole Based Authentication.
  • keyboard_arrow_rightRestrict Access Pages By User Role.
  • keyboard_arrow_rightDynamic Sidebar Menu Based On Role.
  • keyboard_arrow_rightAuthGuard Integrated Pages.
  • keyboard_arrow_rightSeperate Module For Admin, Doctor & Patient.
  • keyboard_arrow_rightAnd meny more...
